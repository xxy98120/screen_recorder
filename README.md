# 桌面屏幕录制工具

一个简洁、美观的桌面屏幕录制软件，使用 Python + PyQt5 构建。

## 功能特性

- **屏幕录制**：录制主显示器或扩展显示器
- **帧率控制**：10-60 FPS 可调
- **鼠标指针**：可选择是否录制鼠标轨迹
- **暂停/恢复**：录制过程中可随时暂停
- **系统托盘**：最小化后后台运行
- **深色主题**：现代炫酷的暗色界面
- **MP4 输出**：直接输出标准 MP4 视频文件

## 安装与运行

### 1. 安装依赖

```bash
cd screen_recorder
pip install -r requirements.txt
```

### 2. 运行程序

```bash
python main.py
```

## 使用说明

1. 选择要录制的显示器（主显示器 / 显示器 2）
2. 调整帧率（推荐 30 FPS）
3. 选择是否显示鼠标指针
4. 点击「开始录制」
5. 录制中可使用「暂停/继续」控制
6. 点击「停止」结束录制，视频自动保存

## 输出格式

- 视频格式：MP4 (H.264)
- 输出目录：默认 `~/Videos`，可自定义

## 技术栈

- **PyQt5** - 跨平台桌面 GUI
- **OpenCV** - 视频编码
- **mss** - 高性能屏幕捕获
- **sounddevice** - 音频支持（预留）
